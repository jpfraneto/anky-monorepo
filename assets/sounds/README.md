# Anky Bell

The bell marks the boundary.

## Files
- `anky_bell.wav` - 7.6 seconds, 32kHz, mono
  - Deep meditative gong bell
  - Spiritual, resonant tone
  - Sustained, fading slowly
  - Ceremonial

## Usage

**At the beginning:** The bell sounds. 8 minutes begins. No backspace.

**During:** The bell marks the passage. A reminder. You are here.

**At the end:** The bell sounds. 8 minutes is over. What you wrote is written.

## Technical
- Format: WAV
- Sample rate: 32000 Hz
- Channels: Mono
- Duration: 7.6 seconds
- Size: 953 KB

## Integration

For iOS app:
```swift
// Play the bell
let audioPlayer = AVAudioPlayer()
let url = Bundle.main.url(forResource: "anky_bell", withExtension: "wav")
audioPlayer = AVAudioPlayer(url: url)
audioPlayer.play()
```
