# Anky Content OS

Anky is a small blue witness for the sentence under the noise.

This repo manages Instagram content for Anky: brand rules, ritual prompts, launch posts, calendar planning, and resonance tracking.

Core promise:

> write for 8 minutes and listen to yourself again.

## Structure

```text

```

## Working Loop

1. Pick the pillar for the day: Awaken, Navigate, or Know Yourself.
2. Choose one doorway prompt.
3. Write the post in Anky's voice.
4. Invite the 8-minute ritual.
5. Track saves, shares, one-word comments, DMs, and ritual completions.
6. Use the resonance score to decide what should become a community rug.

## Core CTA

Give it 8 minutes. Do not make it beautiful. Make it audible.
